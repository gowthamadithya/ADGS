# Documentation

## Class: Calculator

```text
This class provides basic arithmetic operations like addition and subtraction.
```

## Function: add

```text
Adds two numbers.

Args:
a (int): First number
b (int): Second number

Returns:
int: The sum of a and b
```

## Function: subtract

```text
Subtracts one number from another.

Args:
a (int): First number
b (int): Second number

Returns:
int: The difference of a and b
```
